1. Ожидается, что функция: "capitalize(self, string: str) -> str" принимает строку, делает первую букву заглавной
   и возвращает этот же текст. 
   Фактически:
    - нет описания поведения функции, если принять пустую строку. 
    - пустая строка, сторока с прбелами проходят тест
    - нет описания поведения функции(ошибки), если принять число, список, например [a, b]

2. Ожидается, что функция: "trim(self, string: str) -> str" Принимает на вход текст и удаляет пробелы в начале, если они есть
   Пример: `trim("   skypro") -> "skypro". 
   Фактически:
    - нет описания поведения функции, если принять пустую строку. 
    - пустая строка, сторока с прбелами проходят тест
    - нет описания поведения функции(ошибки), если принять число, список, например [a, b]

3. Ожидается, что функция: "contains(self, string: str, symbol: str) -> bool" Возвращает `True`, если строка содержит искомый символ
        и `False` - если нет
        Параметры:
            `string` - строка для обработки
            `symbol` - искомый символ
        Пример 1: `contains("SkyPro", "S") -> True`
        Пример 2: `contains("SkyPro", "U") -> False`
    Фактически:
    - нет ограчения количества искомых символов (symbol):
      если пустая строка на входе, то в результате True
      если в symbol передать строку длиннее строки для обработки (string), то в результате False
